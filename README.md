# Site Yönetim Paneli

Site yöneticileri için kapsamlı, modern arayüzlü ve masaüstü (Electron.js) tabanlı yönetim uygulaması. 
Ayrıca site sakinleri için entegre bir borç sorgulama portalı sunar.

## 🌟 Özellikler
- **Admin Paneli**: Electron.js tabanlı, SQlite veritabanı ile tamamen yerel çalışan masaüstü yönetim arayüzü.
- **Aidat ve Finans Takibi**: Standart aidat tanımlama, ekstra ödeme (demirbaş) takibi ve detaylı tahsilat işlemleri.
- **Etkileşimli Site Haritası**: Site vaziyet planı üzerinden renklendirilmiş (borcu olmayanlar yeşil, borcu olanlar kırmızı/turuncu) interaktif SVG/KML harita görünümü.
- **Sakin Portalı (Web Entegrasyonu)**: Yönetim panelinden tek tıkla GitHub Pages'e güncel verileri yükleyen ve site sakinlerinin kendi borç durumlarını görebileceği şık web arayüzü.

## 🛠 Kullanılan Teknolojiler
- **Masaüstü Uygulaması**: Electron.js, Node.js
- **Veritabanı**: SQLite3
- **Arayüz (Frontend)**: Vanilla JavaScript, HTML5, CSS3 (Modern Glassmorphism tasarım)
- **Harita**: Leaflet.js
- **Yayınlama**: GitHub Pages & Git Integration
